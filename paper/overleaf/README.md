# Overleaf Upload Folder

Upload the contents of this folder as a new Overleaf project.

## Files

- `main.tex`: main LaTeX source file
- `figures/`: place figure PNG files here

## Figure filenames

If you have generated the figures locally, copy them into `figures/` with these exact names:

- `figure_1_refusal_rate.png`
- `figure_2_prompt_level_consistency.png`
- `figure_3_framing_sensitivity_histogram.png`

`main.tex` will still compile without the PNG files. In that case it shows placeholder boxes instead of figures.

## Main file

Set `main.tex` as the Overleaf main document if needed.
